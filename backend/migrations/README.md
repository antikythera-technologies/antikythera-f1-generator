# Database Migrations

This directory contains Alembic migrations for the Antikythera F1 Video Generator database.

## Setup

1. Initialize Alembic (already done):
```bash
alembic init migrations
```

2. Update `alembic.ini` with your database URL or use environment variables.

## Common Commands

```bash
# Create a new migration
alembic revision --autogenerate -m "description"

# Apply all migrations
alembic upgrade head

# Rollback one migration
alembic downgrade -1

# Show current revision
alembic current

# Show migration history
alembic history
```

## Naming Convention

Migration files follow the pattern:
```
001_initial_schema.sql
002_add_character_images.sql
003_add_cost_tracking.sql
```

## Notes

- Always review autogenerated migrations before applying
- Test migrations on a development database first
- Back up production database before applying migrations
